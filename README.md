## 简介
### 什么是进程管理工具？
进程管理工具是一款专业的Windows系统进程监控和管理软件，可以帮助用户查看、管理和控制系统上运行的所有进程。
### 适用人群
- **普通用户**：想了解电脑运行了哪些程序
- **IT技术人员**：需要诊断系统问题或管理进程
- **系统管理员**：批量管理多台电脑的进程
- **开发人员**：调试应用程序时监控进程状态
### 主要功能
- 查看所有运行中的进程及其详细信息
- 结束、挂起和恢复进程
- 隐藏/显示特定进程窗口
- 监控CPU和内存使用情况
- 记录操作日志

## 安装与运行
### 系统要求
- 操作系统：Windows 7/8/10/11
- Python 3.6或更高版本
- 管理员权限
### 安装步骤
1. 确保已安装Python
2. 安装所需库：
   ```
   pip install psutil pyqt5
   ```
3. 下载ProcessManager_app.py文件
### 运行方法
1. 右键点击文件，选择"以管理员身份运行"
2. 或通过命令行：
   ```
   python ProcessManager_app.py
   ```

## 基本功能
### 查看进程列表
- 程序启动后自动加载所有进程
- 每个进程显示：名称、PID(进程ID)、窗口标题(如果有)
### 刷新进程列表
点击"刷新进程列表"按钮或右键菜单中的"刷新"选项
### 查看进程详情
点击列表中的进程，详细信息将显示在日志标签页中，包括：
- CPU和内存使用情况
- 启动时间
- 执行路径
- 命令行参数
### 结束进程
1. 右键点击目标进程
2. 选择"结束进程"
3. 确认操作
*注意：结束系统关键进程可能导致系统不稳定*

## 高级功能
### 隐藏/显示进程
- **隐藏进程**：右键菜单选择"隐藏进程"，该进程窗口将从桌面消失
- **显示进程**：对已隐藏的进程选择"显示进程"
*应用场景*：
- 临时隐藏干扰窗口
- 保护隐私不被他人看到
### 挂起/恢复进程
- **挂起**：暂停进程运行
- **恢复**：让挂起的进程继续运行
*技术说明*：
使用Windows API的DebugActiveProcess函数实现
### 批量操作
- **显示隐藏进程**：勾选"显示隐藏进程"复选框
- **结束所有隐藏进程**：一键清理所有隐藏进程
### 日志功能
- 所有操作自动记录
- 支持清空日志
- 错误信息显示为红色

## 常见问题
### Q1: 为什么有些进程无法结束？
A: 可能是系统关键进程或权限不足，请确保以管理员身份运行本工具。
### Q2: 隐藏的进程去哪里了？
A: 进程仍在运行，只是窗口不可见。可以在"显示隐藏进程"模式下查看。
### Q3: 挂起的进程会影响系统吗？
A: 挂起系统进程可能导致相关功能暂时不可用，请谨慎操作。
### Q4: 如何知道哪个进程占用了大量内存？
A: 查看进程详情中的"内存使用"信息，数值越大占用越多。

## 安全提示
1. 结束进程前请确认其作用，误操作可能导致：
   - 数据丢失
   - 程序异常
   - 系统不稳定
2. 隐藏进程功能不应被用于：
   - 隐藏恶意软件
   - 绕过安全监控
3. 建议操作前：
   - 保存所有工作
   - 了解目标进程的功能
